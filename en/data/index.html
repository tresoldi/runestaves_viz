abstract: >-
  A comprehensive digital database of 891 Swedish medieval and early modern
  perpetual calendars (runestaves/primstavar), containing 243,903 daily
  transcription records with feast days, symbolic notation, and geographic
  metadata. The database includes validated TSV files, a SQLite database (~34
  MB), and phylogenetic analysis data. This release features feast name
  normalization (283 variants → 268 canonical forms) and symbolic notation
  decomposition (29,908 fest_marks → 29,585 symbol instances). The database
  enables computational analysis of vernacular timekeeping practices, cultural
  transmission patterns, and the interplay between ecclesiastical and folk
  traditions in medieval Scandinavia.
